## 🧑‍💻 Running Locally
1. Clone the repository
```bash
git clone [https://github.com/alexandredev3/ignews.git](https://github.com/alexandredev3/image-insight.git)
```

2. Install dependencies
```bash
pnpm install
```

3. Create a `.env` file. See `.env.example` for reference.

4. Start up server!
```bash
pnpm dev
```
